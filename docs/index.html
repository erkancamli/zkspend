zkSpend — MVP

Flow: Connect wallet → Upload receipt → Off-chain worker computes commitment + (later) ZK proof → Call claim() on Campaign.

  1. Deploy VerifierStub and CampaignFactory (Foundry script).
  2. Create a Campaign with params.
  3. Run worker/worker.py on your receipt to get publicInputHash & placeholders.
  4. Call claim() in a simple script or Foundry.

Next: replace verifier with RISC Zero verifier, store encrypted receipt blob on 0G Storage, mint EAS attestation.
